
# 台灣走走旅遊資訊網
<img width="300" src="https://i.imgur.com/gXeZonp.png"/>

這是一個台灣旅遊資訊網站，使用者可以收尋三種主題「景點」、「活動」、「美食」，
也可以依各縣市收尋以上三種主題的資訊內容，進入資訊內頁還可以依景點所在地點收尋到附近的「景點」、「活動」、「美食」三種主題內容。

👉 https://b9111426.github.io/taiwan-travel/


- 交通部TDX API串接資料
- 活動日期篩選
- 資料收尋
- Geohash編碼查詢範圍
- 地圖運用
- 使用js客製化select表單樣式
- jQuery datapicker套用
